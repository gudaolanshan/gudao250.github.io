hexo订阅出现的问题

开始在微博中加入rss

利用npm依赖下载并安装rss插件：

npm install hexo-genetator-feed --save

_config.yml文件添加代码：

Plugins:
- hexo-generator-feed
theme: landscape

添加百度优化插件，让自己的博客更容易检索到：

npm install hexo-generator-baidu-sitemap --save

_config.yml文件添加代码：

baidusitemap:
    path: baidusitemap.xml

安装该博客遇到的坑

第一次使用hexo,nodejs和github上传博客

在这个过程中，安装很顺利，但是卡在上传本地博客到github这一步。

2个问题：


  • 上传不了数据到github：

    • 错误点：原本是

repo: https://github.com/gudaolanshan/gudaolanshan.github.io.git
现在已经没办法使用这种写法了。
解决办法：_config.yml文件中修改为：
deploy:
    type: git
    repo: git@github.com:gudaolanshan/gudaolanshan.github.io.git
    branch: master

<li>github网页无法打开</li>
错误点：git仓库名为gudao250.github.io，但是git要求是前面部分必须和账号名相同。<br>
解决办法：把git仓库重命名为gudaolanshan.github.io
