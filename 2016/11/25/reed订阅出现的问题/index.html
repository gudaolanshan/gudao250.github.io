reed订阅出现的问题

开始在微博中加入res

利用npm依赖下载并安装rss插件：

npm install hexo-genetator-feed --save

_config.yml文件添加代码：

Plugins:
- hexo-generator-feed
theme: landscape

添加百度优化插件，让自己的博客更容易检索到：

npm install hexo-generator-baidu-sitemap --save

_config.yml文件添加代码：

baidusitemap:
    path: baidusitemap.xml
