安装该博客遇到的坑

第一次使用hexo,nodejs和github上传博客

在这个过程中，安装很顺利，但是卡在上传本地博客到github这一步。

2个问题：


  • 上传不了数据到github：



    • 错误点：原本是

    repo: https://github.com/gudaolanshan/helloworld.git

    现在已经没办法使用这种写法了。



    解决办法：_config.yml文件中修改为：

    deploy:

    type: git

    repo: git@github.com:gudaolanshan/gudaolanshan.github.io.git

    branch: master




  • github网页无法打开



    • 错误点：git仓库名为gudao250.github.io，但是git要求是前面部分必须和账号名相同。



    解决办法：把git仓库重命名为gudaolanshan.github.io
